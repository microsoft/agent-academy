# Academia Agentului Copilot Studio

Acest depozit găzduiește un curs în trei părți pentru persoanele care doresc să construiască agenți cu Microsoft Copilot Studio.

Vă rugăm să vizitați Academia Agentului Copilot Studio la [https://aka.ms/agent-academy](https://aka.ms/agent-academy).

Acord de Licență pentru Contribuitori (CLA) care declară că aveți dreptul și, de fapt, acordați drepturile

oferite de bot. Va trebui să faceți acest lucru o singură dată pentru toate depozitele care utilizează CLA-ul nostru.
