# Akademia Agentów Copilot Studio

To repozytorium służy do hostowania trzyczęściowego kursu dla osób, które chcą tworzyć agentów za pomocą Microsoft Copilot Studio.

Odwiedź Akademię Agentów Copilot Studio pod adresem [https://aka.ms/agent-academy](https://aka.ms/agent-academy).
