# 應用深度推理評估候選人適配性及面試準備

--8<-- "disclaimer.md"

## 理論

- 解釋什麼是深度推理
- 討論如何使用深度推理（在代理本身或提示中）
- 討論什麼時候應該使用深度推理

## 實驗

針對面試準備代理，我們將使用深度推理。在這種情況下，我們將對以下內容進行推理：

- 面試問題（是否有需要在面試中深入探討的空白點）
- 公司適配性的提示結果（關於該候選人如何符合公司價值觀的文字描述 + 評分）
- 職位適配性的提示結果（關於該候選人如何符合職位要求的文字描述 + 評分）

這是一個很好的時機來應用學員在之前課程中學到的高級指導。
