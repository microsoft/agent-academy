# Akademija agenata Copilot Studio

Ovaj repozitorij namijenjen je za hosting trodijelnog tečaja za ljude koji žele izraditi agente pomoću Microsoft Copilot Studija.

Molimo posjetite Akademiju agenata Copilot Studio na [https://aka.ms/agent-academy](https://aka.ms/agent-academy).

---
