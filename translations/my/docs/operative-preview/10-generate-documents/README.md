- ဒီမစ်ရှင်မှာ အခန်းကဏ္ဍနှင့် လူကြီးမင်း၏ ရResume အပေါ်မူတည်ပြီး အထူးသီးသန့် အင်တာဗျူးမေးခွန်းများကို ဖန်တီးပေးမယ့် Prompt တစ်ခုကို ဖန်တီးရမည်။
