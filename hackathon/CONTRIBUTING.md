# Contributing to the Agent Academy Live Hackathon

Thank you for your interest! This is a hackathon repository, so contributions are limited to improvements that help all participants.

## ✅ What We Welcome

- **Bug reports** for documentation or submission process
- **Documentation fixes** (typos, clarifications, broken links)
- **Community resources** (helpful guides, tools)
- **Curriculum feedback** via the [Product Feedback](../../issues/new?template=product-feedback.md) issue template

## ❌ What We Don't Accept

- Changes to judging rubric or rules during the hackathon
- Direct edits to hackathon submissions

## 🐛 Reporting Issues

Found a bug? [Open an issue](../../issues/new?template=general-question.md) with:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior

## 📝 How to Contribute

### Documentation Fixes

1. Fork this repository
2. Make your changes
3. Test that everything works
4. Submit a Pull Request with:
   - **Title**: `Fix: [brief description]` or `Docs: [what you improved]`
   - **Description**: What you changed and why

## 📋 Pull Request Guidelines

**Good PR titles:**
- `Fix: Broken link in README`
- `Docs: Clarify submission requirements`
- `Fix: Typo in judging rubric`

**In your PR description:**
- What you changed
- Why it's helpful
- How you tested it

### Review Process

1. **Maintainer review**: Program team reviews your PR
2. **Feedback**: Address any requested changes
3. **Merge**: Once approved, we'll merge your contribution!
